Profile:

My name is David. I graduated from a B.Sc. Mathematics in 2022. There I learnt about several topics; mostly contemporary mathematics. I began coding around 2014 when I was 16 years old. My first programming language was C++. Then in my last year in college I dived into high-level programming languages, particularly Python and JavaScript.

As an student:

  • A first course in basic programming and another one in programming with numerical methods (both in C++)
  • Two courses in computation theory; finite automatons and Turing machines. The foundations of computer science.
  • One course in graph theory; structures, trees and graphs. It has some applications in data science and APIs.
  • One course in linear algebra and another one in multilinear algebra. It can be used for computational graphics and even quantum computing.
  • Three courses in abstract algebra; algebraic structures such as groups, rings and fields.
  • Several freeCodeCamp certifications; Web design, Javascript, Frontend, Backend, Data science, among others.
  • A diplomate in Software Development by MINTIC (Python, basics of Java and Django framework.)

Back in those years I worked in several part-time jobs. Archivistics, teaching and even customer service... I didnt have much time to keep a job in the long term. So when I finished studying, I decided to focus exclusively in learning how to build web and native applications. It took me time to get in the track. But after some months and after finishing some virtual courses on the topic (from which I mostly recommend freecodecamp.org and fullstackopen.com). I was finally able to build software apps by my own. I have found this very enriching.

At the very moment I would like to dive more into Tensorflow and Qiskit
