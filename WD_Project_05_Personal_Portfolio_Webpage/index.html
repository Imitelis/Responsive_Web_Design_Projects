About me:

My name is David. I graduated from a B.Sc. Mathematics in 2021. There I learnt about several topics; mostly contemporary mathematics. I began coding around 2014 when I was 16 years old. My first programming language was C++. Then through my last year in college I dived more into high-level programming languages, mostly Python and JavaScript.

As an student:

  1. 1. A basic programming course and a programming with numerical methods course (C++).
  2. 2. Two computation theory courses; finite automatons and Turing machines. The foundations of computer science.
  3. 3. A graph theory course; structures, trees and graphs. It has some applications in data science and APIs.
  4. 4. A linear algebra course and a multilinear algebra course. It can be useful for computational graphics.
  5. 5. Several courses in mathematical analysis, abstract algebra, general topology, mathematical logics, etc.
  6. 6. Research in an spectrographic analysis group (computer vision) for an academic start-up (ScikitPy, FlaskPy, NextJS).
  7. 7. Various freeCodeCamp certifications; Web design, Javascript, Frontend, Backend, Data science, among others.
  8. 8. A national diplomat in Software Development (Python, basics of Java, MySQL and DjangoPy framework).
  9. 9. FullStack Open certification (ReactJS, NodeJS, MongoDB, Redux, GraphQL, TS, ReactNative, Docker).
  10. 10. At the very moment I would like to emphasize in Tensorflow and Qiskit.
